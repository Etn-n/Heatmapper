<h1>Create Heatmaps of your files</h1>
With just a few clicks

<h2>Dependencies :</h2>
python > 3.13 <br>
Windows only (because of os module) <br>
Modules :<br>
<ul>
<li>subprocess,os,sys,csv (should be installed automatically with python)</li>
<li>PySide6</li>
<li>pandas</li>
<li>matplotlib</li>
</ul>
<br>
To install them all : <code>pip install -r requirements.txt</code>

<h2>You need to get XAN (https://github.com/medialab/xan) and put <code>xan.exe</code> in <code>./data/</code></h2>
